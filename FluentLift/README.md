# FluentLift

## 🇬🇧 English Version

### FluentLift Language Learning Platform

FluentLift is a simple language learning platform developed using **PHP**, **MySQL**, and **localhost (XAMPP)**. The project aims to provide users with an easy and interactive way to improve their English learning experience.

📚 Created as a final project for a university course.

🛠️ Built with **PHP**, **MySQL**, **HTML**, **CSS**, **JavaScript**, and **localhost (XAMPP)**.

✨ Features include user authentication, vocabulary management, progress tracking, and a simple, user-friendly interface.

🚀 Open to future improvements and new features.

Feel free to explore, use, or contribute!
