# 👗 Sana Yakışan

## 🇬🇧 English Version

### Body Type & Style Blog Website

**Sana Yakışan** is a WordPress-based blog website created to provide information about body types, fashion recommendations, and style tips. The website helps visitors discover clothing suggestions that best match different body shapes through informative blog content.

📚 Created as a final project for a university course.

🛠️ Developed using **WordPress**, **PHP**, **MySQL**, **HTML**, **CSS**, and **JavaScript**.

✨ Features include:

* Responsive and user-friendly design
* Blog posts about body types and fashion
* Category and tag management
* Image gallery
* Contact page
* Navigation menu
* WordPress plugin integration
* Easy content management through the WordPress dashboard

🚀 Open to future improvements such as user comments, advanced search, personalized style recommendations, and additional blog categories.

Feel free to explore, use, or contribute!

## 🇹🇷 Türkçe Versiyon

### Vücut Tipleri ve Stil Blog Sitesi

**Sana Yakışan**, farklı vücut tiplerine uygun giyim önerileri, moda tavsiyeleri ve stil ipuçları sunmak amacıyla **WordPress** kullanılarak geliştirilmiş bir blog web sitesidir. Ziyaretçilerin kendi vücut tiplerine uygun kombinleri keşfetmelerine yardımcı olacak bilgilendirici blog içerikleri sunmaktadır.

📚 Üniversite dersi kapsamında hazırlanmış bir final proje ödevidir.

🛠️ **WordPress**, **PHP**, **MySQL**, **HTML**, **CSS** ve **JavaScript** kullanılarak geliştirilmiştir.

✨ Özellikler:

* Responsive ve kullanıcı dostu tasarım
* Vücut tipleri ve moda üzerine blog yazıları
* Kategori ve etiket yönetimi
* Görsel galerisi
* İletişim sayfası
* Kolay kullanılabilir gezinme menüsü
* WordPress eklenti desteği
* WordPress yönetim paneli üzerinden kolay içerik yönetimi

🚀 Kullanıcı yorumları, gelişmiş arama, kişiselleştirilmiş stil önerileri ve yeni blog kategorileri eklenerek geliştirilmeye açıktır.

İnceleyebilir, kullanabilir veya katkıda bulunabilirsiniz!
